About

Hello, my name is Alex. I'm a python dev.

Some years ago my parents bought me a Texas Instruments model TI 84 Plus graphing calculator to use in a trigonometry class. Naturally, I immediately wanted to find out if I could play games on this thing. Delighted as I was to find out the answer was yes, I was curious how exactly this calculator could enable me to play snake while I admittedly should have being doing my homework. Eventually I figured out where in the menus of the device I could find the code that these games ran from. Needless to say my curiosity was piqued. Using the internet I was able to find the documentation for the calculator and I taught myself some simple syntax. Once I started writing programs to assist me with calculations in my trigonometry class, I was hooked. The power of programming became clear to me and I've been in love with it ever since.

For the past couple years I've been studying python specifically, along with some other tangentially related languages as necessary(like the HTML and CSS for this webpage). I've even built a few projects which you can check out here. I'm a fast learner and I am eager to bring my enthusiasm for programming to your next project or job opportunity. I'll be looking forward to hearing from you.

Projects

  • Kivy Pong

    This is a simple pong game with the graphics powered by Kivy: an open source python library. There is an AI opponent and the ball accelerates to a maximum speed as it bounces back and forth. First one to 10 points wins!
    Check it out here.

  • Password Checker

    This program checks a password using the API from haveibeenpwned.com. The UI is powered by PySimpleGui to prompt the user for a password. The output is whether or not the password has been seen in a data breach and how many times it has been seen.
    Check it out here.

  • Twitter Bot

    This program accesses twitter using the Tweepy library to access the Twitter API. It's set up to search tweets based on a keyword then like the 5 most recent tweets with that keyword. However, this can be reconfigured to do any number of things with the twitter API.
    Check it out here.

  • Hacker News Scraper

    This program scrapes Hacker News and pulls and displays the top 10 articles with over 100 votes. The UI is powered by the tkinter library which is the standard python interface to the Tk GUI toolkit.
    Check it out here.

  • File Conversion: JPG to PNG

    This project simply takes a target directory, finds all the JPG files then outputs PNG files to another target directory. Target directories are chosen by command line argument or file dialog if no argument is given.
    Check it out here.

  • PDF Watermarker

    This straightforward yet useful program accepts a pdf file and adds a watermark of your choice to every page. Quick and easy way to watermark many pages of a PDF.
    Check it out here.
